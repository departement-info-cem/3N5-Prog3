  <TabItem value="ModalNavigationDrawer" label="ModalNavigationDrawer">

    La fonction composable `ModalNavigationDrawer` permet de créer un contenant:
    <ul>
        <li>intéragit avec un drawerState qui permet d'ouvrir / fermer le tiroir</li>
        <li>prend un drawerContent qui sera l'objet graphique qui disparaît à gauche</li>
        <li>on va passer le navController au tiroir pour permettre de naviguer sur les clicks</li>
        <li>le contenu du `ModalNavigationDrawer` sera ce qui s'affiche à l'écran</li>
    </ul>

<TabItem value="hamburger" label="Hamburger">

  Le hamburger est l'icône qui permet d'ouvrir le tiroir de navigation (trois barres horizontales, comme le pain, le steak et le pain d'un hamburger).
  Il est généralement placé dans la barre d'action en haut de l'écran.
  <ul>
    <li>la convention veut qu'on mette le hamburger en haut à gauche dans le topbar</li>
    <li>`IconButton` nous permet d'avoir un bouton avec l'icône de hamburger</li>
    <li>drawerState permet d'ouvrir le tiroir avec `drawerState.open()`</li>
  </ul>

<TabItem value="tiroir" label="Le tiroir">

  Le tiroir va être une fonction composable qu'on peut réutiliser dans plusieurs écrans.

  La fonction va être ici ModalDrawerSheet avec des NavigationDrawerItem dedans:
  <ul>
    <li>prend en paramètre le navController pour déclencher les navigations</li>
    <li>la fonction fermerTiroir permet d'intéragir avec le drawerState défini un niveau plus haut</li>
    <li>ModalDrawerSheet : permet de créer un tiroir qui peut être ouvert et fermé</li>
    <li>NavigationDrawerItem : permet de créer un item de menu dans le tiroir</li>
  </ul>
