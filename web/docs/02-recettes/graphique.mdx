# Graphique

<Row>

<Column>

Il existe plusieurs librairies pour afficher des graphiques dans une application Android.
Nous allons utiliser la librairie [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart), mais sachez qu'il existe plusieurs autres librairies qui offrent des fonctionnalités similaires.

Le code de cette recette est disponible [ici](https://github.com/departement-info-cem/3N5-Prog3/tree/main/code/RecetteGraphique).

:::danger Attention!

Assurez-vous d'avoir configuré [Jitpack](./ajout-librairie-jitpack.mdx) avant de commencer.

    Ajouter la dépendance à la librairie `MPAndroidChart` dans le fichier `build.gradle.kts` du module `app`.

    ```kotlin
    dependencies {
        // Autres dépendances...
        implementation("com.github.PhilJay:MPAndroidChart:v3.1.0")
    }
    ```
